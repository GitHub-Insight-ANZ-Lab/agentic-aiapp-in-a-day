How to Play

  1. Register: Enter your name and click Register
  2. Wait: Wait for the tournament to start (referee controls this)
  3. Auto-Play: The client will automatically participate in all 5 rounds
  4. Questions: Client attempts to answer questions automatically
  5. Moves: Client randomly selects Rock, Paper, or Scissors
  6. Results: View your results after each round

Note: Make sure the game server is running on http://localhost:5289
